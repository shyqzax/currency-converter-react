# Currency Converter 💱

[![GitHub Pages](https://img.shields.io/badge/deployed%20on-GitHub%20Pages-blue)](https://shyqzax.github.io/currency-converter-react/)

Простой и удобный конвертер валют с актуальными курсами. Написан на React с использованием хуков (`useState`, `useEffect`, `useContext`), поддерживает поиск по валютам и сохраняет выбор в localStorage.

🔗 **[Смотреть демо](https://shyqzax.github.io/currency-converter-react/)**

## ✨ Возможности
- Конвертация по реальному курсу (ExchangeRate API)
- Фильтрация валют в реальном времени
- Кнопка смены валют местами (swap)
- Сохранение выбранных валют в `localStorage`
- Адаптивный дизайн для мобильных
- Обработка ошибок и индикатор загрузки

## 🛠️ Технологии
- **React** (hooks: `useState`, `useEffect`, `useContext`)
- **Vite**
- **CSS** (адаптивная верстка)
- **ExchangeRate API**
- **GitHub Pages** (хостинг)
